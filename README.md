# pokemon-frontend
 Frontend PokeApi Challenge

Instalación
-----

Ejecutar el comando:

```ruby
npm i 
```

Ejecución
-----

Crear un archivo .env y agregar las variables la conexion a nuestra API: 

```ruby
VITE_API_URL=https://localhost:4000/api
```
Producción
-----

Para crear el build ejecutar el comando:
```ruby
npm run build
```
